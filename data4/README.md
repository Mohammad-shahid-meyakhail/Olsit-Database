# Monitoring Project (Prometheus + Grafana)

## Components
- Prometheus
- Grafana
- Node Exporter
- PostgreSQL Exporter
- Custom Weather API Exporter

## How to Run
docker compose up -d --build

## Dashboards
- Node metrics
- Database metrics
- Weather API metrics

## Features
- 10+ metrics each exporter
- Alerts configured
- Global variable filter
- Real-time refresh
